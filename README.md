# Music School Platform

## About
A responsive music school platform developed using Next.js, Tailwind CSS, and Aceternity UI, with partial TypeScript usage. The application showcases music courses, featured webinars, testimonials, and modern interactive components for an engaging learning experience.

## Features
- Responsive user interface for all devices
- Browse and explore music courses
- Featured webinars section
- Testimonials from students and instructors
- Modern interactive components built with Aceternity UI
- Partial TypeScript usage for type safety
- Styled using Tailwind CSS

## Tech Stack
- Next.js
- Tailwind CSS
- Aceternity UI
- TypeScript (partial)
- JavaScript (ES6+)

## Setup & Installation
Clone the repository:

git clone [repo-url](https://github.com/RukhsarYaqoob/Music-School.git)

Install dependencies:

npm install

Run the development server:

npm run dev
